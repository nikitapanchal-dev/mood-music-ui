# Mood Music UI 🎧

A simple and interactive web app that shows songs based on your mood.

## 🚀 Features
- Select mood (Happy, Sad, Chill)
- Dynamic song list updates
- Background color changes
- Now Playing feature

## 🛠️ Tech Used
- HTML
- CSS
- JavaScript

## 🧠 What I Learned
- DOM manipulation
- Event handling
- Dynamic UI updates

## 📌 Future Improvements
- Add real music playback
- Improve UI design
