# 🎵 Mood-Based Music UI

A simple web app that shows music based on your mood using **HTML, CSS, and JavaScript**.

---

## 🚀 Live Demo

https://nikita-mood-music-app.netlify.app/

---

## 📌 Features

* Select mood (Happy, Sad, Chill)
* Displays songs based on selected mood
* Click a song to show "Now Playing"
* Dynamic UI updates using JavaScript
* Simple and clean design

---

## 🛠️ Tech Used

* HTML
* CSS
* JavaScript

---

## ▶️ Run Locally

1. Download or clone the project
2. Open `index.html` in your browser
